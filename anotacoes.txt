Critérios de exclusão
    1 - Not match reaisRegex (regex that represents a value in brazilian currency)
    2 - Child nodes > 0
    3 - Ser tag <script>
    4 - Set tag <style>

Critérios para comparação
    1 - É o maior item (Suposição)
    2 - Aparece mais x no código (Suposição)
    3 - O valor esá dentro de uma tag <span>
